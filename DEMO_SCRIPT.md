# Demo Video Script (max 4 min)

## Intro (30s)
"Hi! This is Confidential Payment — a privacy-preserving token transfer DApp built with iExec Nox Protocol on Arbitrum Sepolia."

## Architecture (30s)
- Show diagram: User → Frontend → Nox SDK (encrypt) → Smart Contract (FHE operations) → Decrypt
- "Balances are fully encrypted on-chain using FHE. Nobody can see how much you hold or transfer."

## Demo (2.5 min)
1. Connect wallet (MetaMask, Arbitrum Sepolia)
2. Mint tokens (owner mints encrypted amount)
3. Check balance (decrypt to see actual amount)
4. Transfer to another address (encrypted)
5. Show recipient balance updated

## Closing (30s)
"Built with @iexec Nox Protocol. All computations happen on encrypted data — true financial privacy on-chain."
